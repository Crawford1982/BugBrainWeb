# Security policy

This repository publishes the static presence site at [bugbrain.dev](https://bugbrain.dev/). The live site has no login, no forms, no cookies, and no third‑party analytics or tracking scripts.

## Reporting

**Findings about this repository, the GitHub Pages build, or the published HTML/CSS/assets**

Use [GitHub private vulnerability reporting](https://github.com/Crawford1982/BugBrainWeb/security/advisories/new) for the repo if available on your account view, or coordinate via GitHub’s security mechanisms your role supports.

**Domain, DNS, TLS, or impersonation concerns**

Handle through your DNS/registrar and, when appropriate, established coordinated disclosure practices. This policy does not describe unrelated product or program testing.

There is no bug bounty for this brochure site.
